# TelegramBot To-do-list
<b>Информация:</b> <br/>
/start - начало работы с программой, создаем запись о пользователе.<br/>
/menu - вызов меню с кнопками:<br/>
<b>Создать задачу</b>, <b>Список задач</b>, <b>Удалить задачу</b>.<br/><br/>
Кнопка <b>Создать задачу</b> создает задачу, если существует запись пользователя в БД.

Кнопка <b>Список задач</b> открывет inline-menu текущих задач, по нажатию на которые можно сделать задачу выделенной.

Кнопка <b>Удалить задачу</b> удаляет выделенные задачи.

/delete_me - удаляет запись пользователя в БД.
